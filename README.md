## 节点测试

### Ping测试
测量节点延迟，越低越好
- <50ms: 优秀
- 50-100ms: 良好
- 100-200ms: 一般
- >200ms: 较差

### 带宽测试
测量下载速度
- >100Mbps: 优秀
- 50-100Mbps: 良好
- 10-50Mbps: 一般
- <10Mbps: 较差
